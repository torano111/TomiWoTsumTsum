The package contains:

* 6 different animations of bomb with eyes (blink, idle, idleblink, kaboom1sec, kaboom3sec, fire) 4 animations of normal bomb (idle, kaboom1sec, kaboom3sec, fire)
* bomb is moving, blinking, waiting for explosion and exploding
* all pictures PNG
* original AI file so you can edit pictures on your own
* JSON files so you can run the animation in Unity

The animation was created with Spine 3.3.07. If your animation is not running be sure that you have the newest assets.
